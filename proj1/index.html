Part 1: Drawing Single-Color Triangles

This portion of the project required us to rasterize a triangle. We used a point in triangle test on each sample point within the bounds given to us. Each sample's coordinates were tested using the coordinate plus (0.5, 0.5). Our algorithm is no waorse than one that checks each sample within the bounding box of the triangle because that is exactly what we do. We check each sample within the bounding box of the triangle. However, we made this faster by taking advantage of the cache. We reversed the ordering of our for loops start with y-coordinates and then moving along each x.

We rasterized triangles by following these steps:

  • Find the x and y bounds of the triangle by taking the (x_min, y_min) to (x_max, y_max) of the coordinates given.
  • We then iterated through each pixel within the bounds and checked if the point was in the triangle using the inside function provided to us in triangulation.cpp.
  • If the point was in the triangle, we called fill_pixel on the coordinate with the color provided to us.

Part 2: Antialiasing by Supersampling

This portion of the project required us to rasterize a triangle by using the sampling methods described in class. We took sample rate amount of samples where took took sqrt(sample_rate) x-coordinate samples and sqrt(sample_rate) y-coordinate samples. We used a point in triangle test on each sample point within the bounds given to us. Each sample's coordinates were tested using the coordinate plus ((0.5 + sample_number_x)/sqrt(sample_rate), (0.5 + sample_number_y)/sqrt(sample_rate)).

We implemented supersampling by following these steps:

  • Find the x and y bounds of the triangle by taking the (x_min, y_min) to (x_max, y_max) of the coordinates given.
  • We then iterated through each pixel within the bounds and also iterated sqrt(sample_rate) amount of samples in each direction
  • We then checked if the sample was in the triangle using the inside function provided to us in triangulation.cpp.
  • If the sample was in the triangle, we filled our sample_buffer based on the coordinate of the pixel and current sample number with the color provided to us.
