Part 1: Simple Rasterization

This portion of the project required us to rasterize a triangle by using the sampling methods described in class. We used a point in triangle test on each sample point within the bounds given to us. Each sample's coordinates were tested using the coordinate plus (0.5, 0.5)

We rasterized triangles by following these steps:

  • Find the x and y bounds of the triangle by taking the (x_min, y_min) to (x_max, y_max) of the coordinate given.
  • We then iterated through each pixel within the bounds and checked if the point was in the triangle using the inside function provided to us in triangulation.cpp.
  • If the point was in the triangle, we called fill pixel on the coordinate with the color provided to us.
